Settings

Payment Mode

Works with Wallet of Satoshi, Strike, Alby, Blink, ZBD, Primal, and any LNURL-pay wallet. Known domains are compressed to single letters when broadcast — so "alice@walletofsatoshi.com" transmits as "alice@W" (10 chars vs 30).

Domain code table▾

Audio Protocol

Silent: uses ~18–22 kHz — inaudible to most adults, captured fine by phone mics. Best for merchant use.
Audible: standard tones, easier to debug. Use if silent mode isn't working on older phones.

How audio works

Broadcast loops continuously with a short gap between cycles — the listener can tune in at any point and decode the next full packet.

Lightning Address mode transmits ~12–20 chars. LNbits mode transmits the full BOLT11 (~300+ chars, may not fit in one GGWave packet).

Listen Mode
🎤

Receive Lightning invoices broadcast over audio from another LN Soundbox device.

Supports both full BOLT11 invoices and compressed Lightning Address tokens. Decoding is automatic.
